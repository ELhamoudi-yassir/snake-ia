# 🐍 Snake IA — Java Processing

Jeu Snake avec un agent IA autonome capable de jouer sans intervention humaine.

## 🎮 Description

Développement d'un jeu Snake classique enrichi d'une intelligence artificielle 
de contrôle autonome. L'agent prend des décisions en temps réel pour maximiser 
son score tout en évitant les collisions.

## 🤖 Intelligence Artificielle

- Implémentation d'un algorithme de décision autonome
- Optimisation des mécanismes de déplacement et d'évitement
- Plusieurs modes de jeu disponibles

## 🚀 Lancer le projet

1. Installer [Processing](https://processing.org/download)
2. Ouvrir le fichier `World.pde`
3. Cliquer sur le bouton **Run** ▶

## 🛠️ Technologies

- Java Processing
- Algorithmes de décision IA
